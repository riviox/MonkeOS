# Build:
* Pacman (Arch Linux, Manjaro, itp.):
```
sudo pacman -S gcc g++
```
* APT (Debian, Ubuntu, itp.):
```
sudo apt install build-essential
```
* DNF (Fedora, CentOS, itp.):
```
sudo dnf install gcc gcc-c++
```
